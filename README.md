# PassionSeed Agent Skills

This repository contains agent skills for the PassionSeed project.

## Skills

### pathlab-manager
Manages expert interviews, PathLab content, seeds, paths, path_days, path_activities, path_content, and node_content in Supabase.

**Use when:** Viewing or editing expert_profiles, expert_pathlabs, learning content management, or PathLab generation.

## Usage

Install skills into your Droid environment:

```bash
droid skills add pathlab-manager https://github.com/passionseed/ps-skills/tree/main/pathlab-manager
```

Or manually copy the SKILL.md files to your `.factory/skills/` directory.

## Adding New Skills

1. Create a new folder with your skill name
2. Add a `SKILL.md` file with the skill definition
3. Update this README
4. Submit a PR
